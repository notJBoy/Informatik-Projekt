  - Spracheinstellung in Account Modal wird beim Öffnen geladen und vorausgewählt.
  - Sprache kann gehändelt werden (de, en, fr, es, it, nl, pl, sv).

  - API-Verbindungskonfiguration entkoppelt von Festwerten in `api_config.php`.

## Neuer Eintrag: Noteneintrag für vergangene Klassenarbeiten (18.03.2026)

### Zwischenprompts (Entwicklungsprozess):

1. **Hauptprompt**:
   - "Füge hinzu, dass wenn eine Klassenarbeit vorbei ist, man dort eine Note eintragen kann welche dann unter dem Tab Noten erscheint."

2. **Bugfix nach Implementierung**:
   - "Der Button funktioniert nicht"
   - Problem: Fehlendes Hidden Input-Feld `examGradeExamId` im Modal
   - Lösung: Input-Feld hinzugefügt

3. **Konsistenz und Integration**:
   - "passe die eingabe der Note so an dass die Angaben im Tab Noten Tab übereinstimmen und auch in dem tab angezeigt werden können"
   - Resultat: Modal-Felder angepasst (Fach, Punkte 0-15, Gewichtung, Beschreibung)
   - Noten werden direkt in `grades`-Tabelle statt nur in `exams` gespeichert

4. **Duplikat-Prävention**:
   - "Achte darauf, dass man nicht jedesmal einen neuen eintrag erstellt wenn man auf den button drückt sondern der alte von dem neuen überschrieben wird"
   - Resultat: `exam_grade_set.php` prüft und löscht alte Klassennoten vor dem Erstellen neuer

### Umsetzung:
  1. Datenbankfeld `grade REAL` zur `exams`-Tabelle in `backend.py` hinzugefügt.
  2. Migrationspfad für existierende DBs hinzugefügt: `ALTER TABLE exams ADD COLUMN grade REAL`.
  3. Neuer Backend-Endpoint `PUT /exams/{user_id}/{exam_id}/grade` implementiert zum Aktualisieren einer Note.
  4. Neue PHP-Datei `exams/exam_grade_set.php` erstellt, um die Note über die API zu speichern.
  5. JavaScript-Funktionen `openExamGradeModal()`, `closeExamGradeModal()` und `setExamGrade()` in `current_dashboard.php` hinzugefügt.
  6. Modal `examGradeModal` in `current_dashboard.php` mit:
     - Verstecktes Feld: `examGradeExamId`
     - Fach (readonly, vom Examen vorausgefüllt)
     - Punkte (0-15, gleich wie im Noten-Tab)
     - Gewichtung (optional, default 1, min 0.5)
     - Beschreibung (optional, z.B. Datum der Klassenarbeit)
  7. `renderExams()`-Funktion aktualisiert, um:
     - Die eingetragene Note (Punkte) anzuzeigen
     - Für vergangene Examen einen Button "📝" zum Eintragen/Bearbeiten der Note anzuzeigen
  8. `exam_grade_set.php` Logik:
     - Lädt alle bestehenden Noten für den User
     - Prüft zuerst auf alte Klassennote (gleicher Fach + "Klassenarbeit" Beschreibung)
     - Löscht alte Klassennote falls vorhanden
     - Speichert die Note direkt in der `grades`-Tabelle (nicht nur in exams)
     - Verwendet Format wie Noten-Tab (Punkte 0-15, Gewichtung, Beschreibung)
     - Noteneintrag: "Klassenarbeit - [Beschreibung]"
     - Trägt die Note auch als Referenz in exams ein
  9. Validierung: Punkte 0-15, Gewichtung > 0, Fehlerbehandlung auf Frontend und Backend.

  - ✅ Noten erscheinen automatisch im Noten-Tab und beeinflussen den Durchschnitt
